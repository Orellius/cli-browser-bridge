# cli-browser-bridge (Archived)

> Rust-powered browser automation bridge for Claude Code via MCP.

## What This Was

A single-binary MCP server that gave Claude Code browser automation capabilities. Controlled a headless browser for tasks like scraping, form filling, and screenshot capture - all invokable as MCP tools from the terminal.

## Stack

- Rust
- MCP (Model Context Protocol)
- Headless browser automation

## Usage

This project is archived and no longer maintained.
